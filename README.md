## 🚀 Quick Start

### Running Locally

No build process required! This is a **completely static site**:

```bash
# Option 1: Python (built-in on most systems)
python -m http.server 8000

# Option 2: Node.js
npx serve .

# Option 3: VS Code Live Server Extension
# Right-click index.html → "Open with Live Server"
```

Then open `http://localhost:8000` in your browser.

## ✏️ Customization Guide

### Adding New Projects

1. Edit `assets/data/projects.json`
2. Add to the `projects` array:

```json
{
  "id": "unique-id",
  "title": "Project Title",
  "shortDesc": "One-line description",
  "fullDesc": "Detailed description for modal",
  "tags": ["web-dev", "react"],
  "tech": ["React", "Node.js"],
  "image": "./assets/images/project.jpg",
  "imagePlaceholder": "linear-gradient(135deg, #e95420, #77216f)",
  "demoUrl": "https://demo.example.com",
  "repoUrl": "https://github.com/user/repo",
  "year": 2025,
  "role": "Lead Developer",
  "contributions": ["Feature 1", "Feature 2"],
  "featured": true
}
```

3. Add project image to `assets/images/`

### Updating Skills

Edit skill bars in `index.html`. Each has a `data-level` attribute (0-100):

```html
<div class="skill-bar__fill" data-level="85" style="width: 0%;"></div>
```

### Changing Colors

Modify CSS variables in `assets/css/styles.css`:

```css
:root {
  --ubuntu-orange: #e95420;      /* Primary accent */
  --ubuntu-purple: #77216f;      /* Secondary accent */
  --text-primary: #1a1a1a;       /* Light mode text */
  /* ... */
}
```

### Contact Form

Uses [Formspree](https://formspree.io/) for submissions:

1. Create account at [formspree.io](https://formspree.io/)
2. Create form and get ID
3. Update in `index.html` form action & `assets/js/main.js` CONFIG
